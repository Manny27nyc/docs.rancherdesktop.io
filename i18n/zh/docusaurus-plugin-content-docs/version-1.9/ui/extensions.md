**Catalog** 选项卡是 Rancher Desktop 的扩展市场。你可以通过单击扩展来获得更多信息，并直接使用 UI 安装扩展。

#### 搜索

你可以使用**搜索**功能来查询可用的扩展。

#### Name

此按钮可用于安装或卸载扩展。
