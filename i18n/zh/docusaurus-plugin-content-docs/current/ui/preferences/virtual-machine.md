### Memory

分配给 Rancher Desktop 的内存。可选范围取决于你的系统。红色区域表示分配可能会影响系统服务。

### CPU

分配给 Rancher Desktop 的 CPU 数量。可选范围取决于你的系统。红色区域表示分配可能会影响系统服务。
